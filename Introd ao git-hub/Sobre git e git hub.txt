O git é um sistema de controle de versão de códigos. Gerencia ramificações feitas 
por diferentes pessoas de um time de modo a manter a ordem e integração de todas as partes. Ou seja,
 mesmo que várias pessoas estejam trabalhando em um programa ao mesmo tempo, ainda é possível 
controlar as mudanças para evitar erros.

Qual a diferença entre Git é GitHub?

git é um software VCS local que permite aos desenvolvedores salvar snapshots de seus 
projetos ao longo do tempo. Geralmente é melhor para uso individual. 
GitHub é uma plataforma baseada na web que incorpora os recursos de controle de versões 
do git para que possam ser usados colaborativamente.
